EatSnap · 拍照存 iCloud

手机端独立采集。图片直接写入 iCloud Drive，由 Mac 端扫描识别。
点击下方「开启摄像头」开始
使用说明
1. 直接「开启摄像头」拍照，或点「从相册选择」挑已有照片
2. 选好后点「保存到 iCloud」，系统会弹出分享面板
3. 选择「存储到文件」→ 找到 iCloud Drive/EatSnap/ 根目录
4. 文件名已编码日期与餐别（如 2026-07-08_早_e3b0_153326.jpg），Mac 端会自动归位
5. Mac 端跑一次 python -m server identify 即可识别
单 HTML，file:// 打开或扔到任意静态托管都能用
